# Battleships Java Game

A Battleships game developed in Java using an MVC-style structure, with both command-line and GUI components.

This project was originally created as university coursework and later cleaned up and published as a portfolio project to showcase object-oriented programming, Java application structure, and game logic implementation.

## Features

- Classic Battleships gameplay
- Java-based implementation
- MVC-style project structure
- Command-line and GUI-related components
- Ship placement and board logic
- Basic test coverage included
